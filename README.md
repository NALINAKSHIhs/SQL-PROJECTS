# SQL-PROJECTS
## Library Management System using SQL
This project showcases the Library Management System using SQL. It covers the creation and managing database tables, implementing CRUD operations, and executing advanced queries. The purpose is to illustrate practical skills in database design, data manipulation, and complex querying techniques.
### OBJECTIVE
* **Set up Database**: Create tables and upload data for branches, employees, members, books, issued status, and return status.
* **CRUD Operations**: Implement Create, Read, Update, and Delete functionality to manage library records.
* **CTAS Usage**: Apply Create Table As Select to generate new tables based on query results.
* **Stored Procedures**: Develop reusable procedures to automate repetitive tasks and encapsulate business logic.
* **Advanced Queries**: Develop complex SQL queries to analyze data, generate reports, and retrieve specific insights.
